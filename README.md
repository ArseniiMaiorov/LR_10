# Менеджер заметок на Qt/QML

Полноценное приложение для создания и хранения заметок с поддержкой блокнотов, вложений (изображения, аудио, видео и любые файлы), оформления подписки и экспорта заметок в текстовый файл. Проект рассчитан на Qt 6.10.1 (минимум 6.8) и комплект Desktop Qt 6.10.1 (shared MinGW64).

## Структура проекта
- `notes.pro` — файл сборки qmake.
- `src/main.cpp` — точка входа, регистрирует `DataStore` в QML.
- `src/datastore.{h,cpp}` — сохранение/загрузка данных в JSON в каталоге `QStandardPaths::AppDataLocation`, экспорт заметок.
- `qml/main.qml` — главное окно с управлением блокнотами, заметками, вложениями, подпиской и экспортом.
- `qml/components/AttachmentDelegate.qml` — отображение вложений и открытие файлов.
- `qml/components/SubscriptionPanel.qml` — панель управления подпиской.
- `qml/qml.qrc` — ресурсный файл с QML.

## Сборка
```bash
qmake notes.pro
make
```
Запуск исполняемого файла `NotesApp` запустит приложение.

### Windows MinGW ошибки линковки
Если компоновщик сообщает об отсутствии статических библиотек форматов изображений (например,
`-ltiff`, `-lwebp` или `-ldeflate`), убедитесь, что сборка выполняется против
динамических Qt-библиотек. В `notes.pro` явно исключены флаги `static`/`staticlib`,
поэтому используйте shared‑комплект Qt (например, "Desktop Qt 6.10.1 MinGW 64-bit")
и не добавляйте `CONFIG += static` вручную.

## Возможности
- Создание, переименование и удаление блокнотов и заметок.
- Редактирование текста заметок с автоматическим сохранением и отметкой времени.
- Добавление вложений любых типов через диалог выбора файла, просмотр вложений.
- Оформление/обновление подписки с хранением плана и даты окончания.
- Экспорт выбранной заметки в `.txt` через диалог сохранения.

Данные автоматически сохраняются в файле `notes.json` в папке приложения (`AppDataLocation`).
